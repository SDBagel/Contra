Moderation

Contra is an extensive platform, and so moderation tools are naturally required. Contra features a powerful set of moderation tools baked in.

Articles

The articles dashboard highlights all articles posted on the site, public and hidden. The list is sortable and articles can be approved, denied, taken down, deleted permanently, and viewed.


A separate page filters only articles that require approval.

Comments

The comments dashboard highlights all comments posted on the site. The list is sortable and comments can be approved, deleted permanently, and viewed with the attached article.


A separate page filters only comments that require approval.

Images

The images dashboard highlights all user uploaded images with a preview and ID. Images can be deleted permanently.

Tickets

The tickets dashboard shows all support tickets. These can be viewed and then immediately deleted because no one cares about support tickets.

Users

The users dashboard shows all users who have an account with a obfuscated email address and actions, like viewing their articles, comments, and tickets.


Users with a verified email can only be banned, if the accounts were to be deleted a new one could be made. Users without a verified email can be deleted directly.


A separate page shows all the users with roles. User roles can be managed from the user dashboard, and roles can be managed from the roles dashboard.

Roles

The roles dashboard shows all user-assignable roles that exist. Any roles added do not add functionality, only the Staff and Administrator roles grant power.
